# 張小姐日系服飾網站

## 專案介紹
本專案為日系服飾電商網站，提供商品展示與購物功能，並支援手機與電腦瀏覽（RWD）。

## 網站連結
https://gigi0516.github.io/clothing-site/

## 功能
- 商品展示
- 購物車（前端模擬）
- 多語系（中 / 英）
- 響應式設計（RWD）

## APP
本專案已開發Android APP，使用WebView技術，將網站封裝成APP。

## 技術
- HTML / CSS / JavaScript
- GitHub Pages
- Android Studio（APP）
